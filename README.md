# germeon_app
Репозиторий с веб сервисом для упрощения рабочих процессов сотрудников ООО Гермеон.


для установки модулей:
pip install -r requirements.txt

ВАЖНО!
В app/settings/ нужно создать файл .env и прописать внутри него данные:
BOT_TOKEN = Токен_тг_бота
BASE_SITE = URL_страницы сайта
TG_ADMIN_ID = ID_своего_телеграма
